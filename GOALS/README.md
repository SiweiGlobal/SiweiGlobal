# GOALS

## 目录用途

记录长期目标、年度目标和阶段目标，并明确区分目标、计划和实际进展。

## 适合记录的内容

- 希望达到的长期方向
- 年度或阶段性目标
- 判断目标是否完成的简单标准
- 与目标相关的计划、进展和调整说明

## 不适合记录的内容

- 没有目标关联的日常任务清单
- 只有愿望、没有范围或判断标准的口号
- 项目的详细过程记录

## 推荐的文件命名方式

长期目标使用 `长期目标.md`，年度目标使用 `YYYY-年度目标.md`，阶段目标使用 `YYYY-Qn-阶段目标.md`。

## 简单示例

`2026-年度目标.md`：分别列出年度目标、实现计划、当前进展和必要的调整。

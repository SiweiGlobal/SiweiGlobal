# JOURNAL

## 目录用途

按日期记录当天或当周发生的事情、完成的工作、个人思考和下一步计划，保留过程性的真实记录。

## 适合记录的内容

- 当天或本周完成的工作
- 遇到的问题和临时思考
- 尚未完成的事项与下一步计划
- 值得以后回看的生活或项目片段

## 不适合记录的内容

- 已经整理成熟、可长期复用的方法
- 项目的完整介绍或具体业务数据
- 只需执行一次的零散提醒

## 推荐的文件命名方式

- 日记：`YYYY-MM-DD-主题.md`
- 周记：`YYYY-Www-周记.md`

## 简单示例

`2026-07-15-RFQ整理进展.md`：记录当天完成的整理工作、发现的问题和明天准备继续处理的事项。

# INSPIRATION

## 目录用途

记录从文章、视频、网站、产品、图片、对话、展览或现实观察中获得的启发，并区分原始来源与个人理解。

## 适合记录的内容

- 来源名称、链接、作者或观察场景
- 原始内容的简要摘要
- 自己受到的启发和联想到的方向
- 可能用于产品、创作或研究的元素

## 不适合记录的内容

- 未注明来源的大段转载内容
- 已经形成完整方案的项目计划
- 只有收藏链接、没有任何个人说明的记录

## 推荐的文件命名方式

使用 `YYYY-MM-DD-来源或主题.md`。

## 简单示例

`2026-07-15-瓷器展览中的蝴蝶纹样.md`：分别记录展览来源、观察到的内容和自己的设计联想。

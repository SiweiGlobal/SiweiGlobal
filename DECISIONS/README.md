# DECISIONS

## 目录用途

记录重要决定的背景、可选方案、最终选择、决定原因和后续影响，方便以后理解当时为什么这样选择。

## 适合记录的内容

- 会影响项目或长期方向的重要选择
- 决定前考虑过的主要方案
- 最终选择及其理由
- 后续结果和需要复查的时间点

## 不适合记录的内容

- 无需解释的日常小选择
- 尚未做出选择的普通想法
- 只记录结论、不说明背景的重要决定

## 推荐的文件命名方式

使用 `YYYY-MM-DD-决定主题.md`。

## 简单示例

`2026-07-15-项目采用独立仓库.md`：记录可选方案、最终决定、原因和对后续管理的影响。
